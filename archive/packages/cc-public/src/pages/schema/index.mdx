# Website Schema

A graph of entities, paired by relationships, arranged into paths.


Entities are independent things:

- [Data](./data.schema)
  - [Dataset](./data/datasets.schema) - a body of structured information
  - [DataCatalog](./data/catalogs.schema) - a collection of Datasets
- [Code](./code.schema)
  - [Notebooks](./notebooks.schema) - literate programming content combinging prose and SoftwareSourceCode
  - [Projects](./projects.schema)
- [Applications](./apps)
  - [Tools]()
- [People](./people)
  - [Organizations](./orgs)
- [Media](./media)
  - [Courses](./courses)
  - [Articles](./articles)
  - [Books](./books)
  - [Videos](./videos)
- [Events](./events)

Relationships pair entities with shared information:

- [Data](./data)
  - [Data_Data]()
- [Code](./code)
  - [Code_Code]()
- 

Paths:

- [Data](./data)
- [Code](./code)
  - [Stack](./stacks)
